# clglPlayer

grab video by ffmpeg, and process image by OpenCL
then show image by OpenGL

### requirement
  - openGL
  - openCL
  - ffmpeg lib

build this project on unix environment
```sh
  $ mkdir build
  $ cd build
  $ cmake ..
  $ make
```

build this project on windows environment
```sh
  $ mkdir build
  $ cd build
  $ cmake ..
  build project by visual studio
```
